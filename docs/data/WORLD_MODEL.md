## Summary (from §31 header)

The PRECIPICE world model describes the simulation layer: how compound tiers map to difficulty, the thematic narrative of the Precipice facility, and the five starter compounds that seed BFS reachability.

## Starter compounds (BFS roots)

All player-accessible compounds must be reachable via BFS from these five:
- Hydrogen Sulfide (T1)
- Ammonia (T1)
- Methane (T1)
- Carbon Dioxide (T1)
- Water (T1)

BFS reachability validated by DataIntegrity test suite (Phase 0).
